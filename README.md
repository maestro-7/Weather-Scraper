# Weather-Scraper

Django web App that scrapes data from google.com and shows on weather page.

 **Screenshots in repo**
 - home.png 
 - weather.png 
 
## Requirements to install
1) Python
2) Django
3) bs4 - Beautiful Soup
4) requests
5) lxml

## Steps to run

- **Clone the repo**
- **Intall requirements**
- **run "python manage.py runserver"**
